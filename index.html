Breve tutorial sobre como utilizar la herramienta DPLOY

¿Qué es DPLOY?

DPLOY es un herramienta que sirve para hacer deployment vía FTP/SFTP y esta construida en node.js

Instrucciones de uso

  1. Instalar DPLOY de manera global, y sus dependencias, mediante: 
    
npm install dploy -g
    Dependiendo de tu instalacion de NPM puede ser necesario utilizar el comando "sudo npm install dploy -g"
  2. Configura DPLOY en tu proyecto 
          
cd TuProyecto/
dploy install
  3. DPLOY creará el archivo "dploy.yaml" el cual deberás llenar con la siguiente configuración usando los datos de tu servidor. 
          
server_name:
  host: ftp.myserver.com
  user: user
  pass: password
  path:
    local:
    remote: httpdocs/TuProyecto
  4. Recomiendo agregar el archivo "dploy.yaml" al archivo ".gitignore", ya que contiene información importante sobre tu servidor y no queremos que quede accesible.

    Una vez completado este paso importante haremos un "commit" ya que es necesario para hacer el deployment. 
          
git init
git add .
git commit -m "Initial commit"
  5. Por ultimo corre el siguiente comando para hacer el "deployment a tu server 
          
dploy server_name
    Es necesario haber creado la carpeta remota en tu servidor antes de hacer el deploy
